# rory-salesforce-claude-skills

A collection of Claude Code skills for Salesforce Commerce demos, built and maintained by [@rory-wickham](https://github.com/rory-wickham).

Skills are organized by Salesforce product area. Each folder contains one or more `.md` skill files plus example scripts where relevant.

## Installation

Clone this repo into your Claude Code commands directory:

```bash
git clone https://github.com/rory-wickham/rory-salesforce-claude-skills.git ~/.claude/commands/salesforce
```

Skills are then invocable by path, e.g.:

```
/salesforce/retail-cloud/retail-cloud-scrape
```
